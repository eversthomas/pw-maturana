Output Format Guide

Given the following CSS/SCSS/SASS input:


/*! Comment */
.navigation {
ul {
line-height: 20px;
color: blue;
a {
color: red;
}
}
}

.footer {
.copyright {
color: silver;
}
}
Expanded

Will ouput the following:


/*! Comment */
.navigation ul {
line-height: 20px;
color: blue;
}
.navigation ul a {
color: red;
}
.footer .copyright {
color: silver;
}
Nested

Will ouput the following:


/*! Comment */
.navigation ul {
line-height: 20px;
color: blue; }
.navigation ul a {
color: red; }

.footer .copyright {
color: silver; }
Compact

Will ouput the following:


/*! Comment */
.navigation ul { line-height:20px; color:blue; }

.navigation ul a { color:red; }

.footer .copyright { color:silver; }
Compressed

Will ouput the following:


/* Comment*/.navigation ul{line-height:20px;color:blue;}.navigation ul a{color:red;}.footer .copyright{color:silver;}
Crunched

Will ouput the following:


.navigation ul{line-height:20px;color:blue;}.navigation ul a{color:red;}.footer .copyright{color:silver;}
